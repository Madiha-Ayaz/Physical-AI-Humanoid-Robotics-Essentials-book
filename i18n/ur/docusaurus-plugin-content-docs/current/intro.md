# فزیکل اے آئی اور ہیومنائڈ روبوٹکس

**13 ہفتوں کا جامع کورس**

## کورس کا جائزہ

اس کورس میں آپ سیکھیں گے:

- **فزیکل اے آئی کی بنیادیں**: روبوٹکس کو اے آئی کے ساتھ ملانا
- **ROS 2**: جدید روبوٹ آپریٹنگ سسٹم
- **سمیولیشن ماحول**: Gazebo، Unity، اور NVIDIA Isaac Sim
- **ہیومنائڈ روبوٹکس**: دو پیروں پر چلنا، توازن، اور کنٹرول
- **وژن-لینگویج-ایکشن سسٹمز**: ملٹی موڈل اے آئی

## سیکھنے کے نتائج

اس کورس کو مکمل کرنے کے بعد، آپ قابل ہوں گے:

1. ROS 2 کا استعمال کرتے ہوئے تقسیم شدہ روبوٹک سسٹم تیار کرنا
2. 3D سمیولیشن ماحول میں روبوٹس بنانا اور ٹیسٹ کرنا
3. NVIDIA کے GPU-accelerated tools استعمال کرنا
4. ہیومنائڈ روبوٹ کنٹرول الگورتھم تیار करना
5. VLA سسٹمز کو لاگو کرنا

## پیشگی تقاضے

- Python پروگرامنگ کی بنیادی سمجھ
- Linear algebra اور calculus کی واقفیت
- Ubuntu/Linux کمانڈ لائن کا تجربہ (مددگار لیکن ضروری نہیں)

## کورس کا ڈھانچہ

| ماڈیول | عنوان | مدت | موضوعات |
|--------|-------|------|---------|
| 1 | بنیادیں | 5 ہفتے | فزیکل اے آئی، ROS 2، ٹولز سیٹ اپ |
| 2 | سمیولیشن | 3 ہفتے | Gazebo، Unity، سینسر ماڈلنگ |

| 4 | ہیومنائڈز اور VLA | 3 ہفتے | دو پیروں پر چلنا، VLA، کیپ سٹون |
